# COVID-19 & Population Analytics Dashboard

## 📌 Project Overview

The "COVID-19 & Population Analytics Dashboard" is an interactive Excel-based data analytics project designed to provide meaningful insights into the impact of COVID-19 across different countries and regions. The dashboard combines pandemic-related metrics with demographic and healthcare indicators to help users analyze trends, compare regions, and evaluate healthcare readiness.

Built using advanced Excel features such as **Pivot Tables, Pivot Charts, Slicers, Timelines, Conditional Formatting, and Dashboard Design techniques**, the project transforms raw data into an intuitive and visually appealing analytical solution.


## 🎯 Objectives

* Analyze COVID-19 cases, deaths, recoveries, and active cases.
* Track vaccination progress across countries and regions.
* Compare population demographics and density.
* Evaluate healthcare infrastructure indicators.
* Enable interactive exploration through filters and timelines.
* Present data-driven insights in a single-page executive dashboard.

## 📊 Dashboard Features

### KPI Cards

The dashboard includes key performance indicators (KPIs) to provide a quick overview of pandemic statistics:

* Total Population
* Total COVID-19 Cases
* Total Deaths
* Recovery Rate
* Active Cases
* Average Vaccination Rate

### Interactive Visualizations

#### 📈 COVID-19 Cases Trend

Tracks the growth and progression of COVID-19 cases over time.

#### 💉 Vaccination Trend

Monitors vaccination progress and coverage across selected regions.

#### 🌍 Top Countries by COVID-19 Cases

Identifies countries with the highest number of reported cases.

#### ⚠️ Top Countries by COVID-19 Deaths

Highlights regions most impacted by COVID-related fatalities.

#### 👥 Population by Country

Compares population distribution across countries.

#### 💉 Vaccination Rate by Country

Analyzes vaccination coverage and performance by country.

#### 🏙️ Population Density by Country

Examines population concentration and its potential impact on disease spread.

#### 💰 GDP vs Population Analysis

Explores the relationship between economic indicators and population size.


## 🔍 Interactive Filters

Users can dynamically filter dashboard data using:

* Country
* State/Region
* Age Group (Derived from Median Age)
* Date Timeline

These filters allow detailed exploration and comparison of specific regions and time periods.


## 📈 Key Insights Generated
